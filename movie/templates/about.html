About This Application, {{ name }}

This Movie Reviews application was developed by Juan Simón Ospina Martínez as part of Taller 1 for the ST0251 course.

This application allows users to:

  • Search for movies and view their details.
  • Add, edit, and delete reviews (once logged in).
  • View latest news related to movies.

All data (movies, reviews, and news) is managed through a database and Django's admin interface.
